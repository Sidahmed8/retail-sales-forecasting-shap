# Retail Sales Forecasting with SHAP Explainability

## Project Overview

This project focuses on retail sales forecasting using machine learning models and explainability techniques.

The goal is not only to predict future sales, but also to understand the main factors that influence sales performance using SHAP analysis.

---

## Business Problem

Retail companies need reliable sales forecasts to optimize inventory, pricing, marketing campaigns and operational planning.

A forecasting model can help answer questions such as:

- What are the expected future sales?
- Which variables have the strongest impact on sales?
- How can the business improve decision-making using predictive analytics?

---

## Objective

The objective of this project is to:

- Analyze historical retail sales data
- Clean and prepare the dataset
- Create useful predictive features
- Train machine learning forecasting models
- Evaluate model performance
- Use SHAP to interpret model predictions
- Identify the main sales drivers
